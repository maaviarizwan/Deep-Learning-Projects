# Customer Churn Prediction using ANN

## Overview
This project predicts whether a bank customer will leave the bank using an Artificial Neural Network (ANN).

## Dataset
- Churn Modelling Dataset

## Technologies
- Python
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- Matplotlib
- Kaggle Notebook

## Model
- Input Layer: 11 features
- Hidden Layer: 3 neurons
- Output Layer: 1 neuron (Sigmoid)

## Results
- Binary Classification
- Customer Churn Prediction
